# ODS 16 — Paz, Justiça e Instituições Eficazes

Site acadêmico sobre o Objetivo de Desenvolvimento Sustentável 16 da ONU,
com apresentação do tema, problemas sociais associados e soluções práticas.

## Arquivos
- `index.html` — estrutura do site
- `style.css` — estilos visuais
- `script.js` — interações (menu, acordeão, contadores, animação da balança)

## Como publicar no GitHub Pages (link para o professor abrir)

1. Crie um repositório novo no GitHub (ex: `ods16-projeto`), público.
2. Envie os 3 arquivos (`index.html`, `style.css`, `script.js`) para esse
   repositório — pelo site do GitHub mesmo: **Add file → Upload files**,
   arraste os arquivos e clique em **Commit changes**.
3. Vá em **Settings → Pages** (no menu lateral do repositório).
4. Em **Branch**, selecione `main` e a pasta `/root`, depois **Save**.
5. Espere cerca de 1 minuto. O GitHub mostrará o link do site, algo como:
   `https://seu-usuario.github.io/ods16-projeto/`
6. É esse link que você compartilha com o professor.

Nenhuma senha ou login precisa ser compartilhado com terceiros para isso —
o próprio dono da conta faz o upload direto pela interface do GitHub.
